# pi-anomaly-simulation
Simulation code for "Finite-Size Spectral Dependence on the Irrational Modulation Parameter in Off-Diagonal Aubry-André Systems" (Downey Jr, 2026). Compares π, φ, √2, e as Aubry-André modulation parameters. Run pi_anomaly_quickstart.py to verify core result in 30 seconds. Requires numpy, scipy, matplotlib.
